Install

Simply download and install the .deb package!

Alternatively, ensure all needed python modules are installed. For Ubuntu 10.10 

$ sudo apt-get install python-simpleparse python-cairo python-rsvg
should be sufficient. Then download the package and extract it and run 
$ sudo python setup.py install
in the source package folder. You will need to get your own sgf files and put them in the /usr/share/gogames-screensaver/sgf folder, or configure the screensaver to look in another folder with gogames-screenaver -c

=======

A gnome-screensaver theme to replay go games from sgf files. Can use both local files and a variety of online sources. Currently only tested under ubuntu 10.10 with python 2.6 (also runs under windows 7, but no installer exists, and the installation process is somewhat involved).

Dependencies

python 2.6
python-cairo
python-gnome2
python-rsvg
python-simpleparse

Install

If you have a 64 bit version of Ubuntu, just download the .deb package and install it! Incidentally, if anyone would like to volunteer to help with packaging for different systems, it would be most welcome - I am pretty ignorant of the whole packaging processes and only managed to produce this deb after much painful trial and error.
